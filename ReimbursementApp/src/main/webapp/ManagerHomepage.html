Welcome manager

This will tell you a little about what the Manager Homepage does

Pending Reimbursements

In the "Pending Reimbursements" section managers will be able to view all pending requests from all employees and have the abilty to resolve requests. Managers will need to fill out a form writing in the Reimbursement ID and selecting an option to approve or deny request.

Reimbursement Receipts

The "Reimbursement Receipts" section displays all resolved reimbursements and will update upon every resolved pending reimbursement.

All Employees

The "All Employees" section displays the information of all employees. As a manager, you will be responsible for using this information as an aid in resolving reimbursement requests.

Search Employee Requests

The "Search Employee Requests" section allows managers to search for a particular employee ID, which will display a table containing all reimbursement request from that employee.
