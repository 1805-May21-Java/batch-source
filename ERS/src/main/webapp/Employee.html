View Pending

View Pending lets the employee sees all pending reimbursement requests that he or she has but yet to be resolved.


View Resolved

View Resolved lets the employee sees all resolved reimbursements that is approved by a manager.

View Information

View Information lets the employee see all the information that has been registered on this site.

Update Information

Update Information allows the employee to add anything that is needed to update his/her information, can leave the unchanged parts blank.

Submit Request

Submit Request Allows the employee to submit any reimbursements to the employee.

He or she will also receive an email when the reimbursement request has been completed.
