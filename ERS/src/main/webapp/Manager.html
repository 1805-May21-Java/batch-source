View Pending

View Pending lets the manager sees all pending reimbursement requests that needs to be resolved.


View Resolved

View Resolved lets the manager sees all resolved reimbursements that is approved by a manger, and whom the manager that approved it.

View Employee

View Employee lets the manager sees all employees that has registered for this ERS website, and lets the manager choose to see any employee's reimbursement requests.

Employee Reimbursement

Employee Reimbursement shows the reimbursement of any particular employee, must click the button at View Employee's correct row first.

Temporary Employee

Temporary Employee Allows the Manager to create a temporary account for any person, and update it to the database.

It will also send an email to the email that has been signed up containing the username and password of that account.
